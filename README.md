# WedgeOps

All-in-one operating system for independent wedding planners. AI run-of-show docs, a unified vendor CRM, budget tracking, and a client portal — replacing HoneyBook + Airtable + five spreadsheets.

## Run locally with Docker

```bash
docker compose up --build
```

Then open http://localhost:3000.

## Run locally without Docker

```bash
npm install --include=dev
npm run dev
```

## Stack

- Next.js 16.2 (App Router, standalone output)
- React 19
- Tailwind CSS v4
- TypeScript
- Dockerized for local + container deploy
